For Linux, Windows or MAC users:
Download all the files from "python script" folder and run "gui_app.py" in a python console. 
The software was developed in python 3.11 and some of the next dependencies can be required: PyQT5, MatplotLib, PYVISA, serial.

For Lazy windows users:
Download all the files from "Windows Compiled" folder and execute "gui_app.exe". 

NOTE: "data" folder should exist so there is a folder for the .csv files if they wanted to be recorded.
